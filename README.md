# Avito Category Classifier

Классификация категорий объявлений на основе текста (заголовок + описание) с использованием классических методов машинного обучения.

Проект выполнен для портфолио при подаче на **Avito Data Science Bootcamp**.

---

## Задача

Предсказать категорию товара по тексту объявления на основе датасета **Avito Category Prediction** с Kaggle. Для ускорения обучения используется сэмпл из 100 000 строк (train) и 20 000 строк (test).

---

## Данные

- Источник: [Avito Category Prediction on Kaggle](https://www.kaggle.com/competitions/avito-category-prediction)
- Целевая переменная: `Category` (кодируется LabelEncoder)
- Входные данные: объединённое поле `title + description`

---

## Подход

### 1. Предобработка
- Объединение заголовка и описания в одно текстовое поле.
- Заполнение пропусков в описании.

### 2. Признаки
- **TF‑IDF** с униграммами и биграммами, 15 000 наиболее частотных терминов (в финальной версии - 5 000 для стабильности GPU).
- **Числовые признаки** (извлекаются из текста):
  - Длина текста (`text_len`)
  - Количество слов (`word_count`)
  - Количество уникальных слов (`unique_words`)
  - Количество заглавных букв (`capital_letters`)
  - Количество цифр (`digits`)
  - Количество знаков препинания (`punctuation`)

Все признаки объединяются в одну разреженную матрицу (sparse matrix) для эффективного хранения.

### 3. Модели
Сравнивались три алгоритма:
- **Logistic Regression** (baseline): мультиклассовая логистическая регрессия.
- **LightGBM**: градиентный бустинг с оптимизированными гиперпараметрами (500 деревьев, глубина 8, регуляризация).
- **CatBoost**: градиентный бустинг с поддержкой категориальных признаков (в данной версии использовались только TF‑IDF и числовые признаки).

Все модели оценивались на валидационной выборке (20% от train, стратифицированный сплит).

---

## Результаты

| Модель                | Accuracy | F1‑macro | F1‑weighted | ROC‑AUC (macro) |
|-----------------------|----------|----------|-------------|-----------------|
| Logistic Regression   | 0.2916   | 0.0327   | 0.1879      | 0.6270          |
| **LightGBM**          | **0.7600** | **0.5579** | **0.7477**  | **0.9654**      |
| CatBoost (GPU)        | 0.6224   | 0.3609   | 0.5798      | 0.9158          |

**LightGBM** показывает точность **76%** и F1‑macro **0.558**, что является хорошим результатом для задачи с сильным дисбалансом классов и модели обученной на сэмпле.
